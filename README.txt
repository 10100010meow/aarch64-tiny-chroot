# AArch64-Tiny-Chroot
A very basic Linux environment for chroot (AArch64 architecture).
Available software should be enough for compiling kernel or something else.

## Installed software
* archivers/compressors
* autotools
* bash
* bc/dc
* bison/yacc
* busybox
* coreutils
* curl
* diffutils
* flex
* gcc/g++/binutils
* gettext
* git (only basic tools)
* intltool
* libtool
* nano
* openssh client
* patch
* perl
* rsync
* shellcheck
* strace
* texinfo
* tmux
* and all needed libs + headers
